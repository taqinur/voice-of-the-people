1. Difference between var, let and Const

a) var is global scoped. Const and let is local scoped.
b) const can't be used in same scope. Var can be used and let can be updated only.
c) let and const are newer and more standard form in Javascript.

2. Difference between arrow and regular function

a) Arrow functions can't be used as constructor but regular function can be used.
b) Arrow funtions are implicitly returned where regular functions don't.
